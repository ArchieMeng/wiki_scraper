# Wiki scraper

Wiki scraper is a web spider to explore articles on [Wikipedia](https://www.wikipedia.org/).

### Dependency
- Python 2.7
- collections
- BeautifulSoup
- urllib
- re
- pickle
- smtplib
- email
- psutil

### Installation
1.install dependences
2.clone this project
### Usage
write emaill address and password to passwd.p using pickle.
Then
```sh
$ python main.py
```
